# Aha! basics you need (5–7 minutes)

You only need a few concepts to build a beginner roadmap:

## The minimal “records” chain
A common structure is:
- Goals / initiatives (why)
- Releases (when)
- Features (what)

You can build a roadmap with just releases + features, but linking to strategy helps later.

### Visual reference (Aha! public knowledge base)
Read: **How to build a roadmap in five steps**

## Where you’ll work
In most accounts you’ll use:
- **Releases** area to create a release
- **Roadmaps → Features** to visualize features across releases
