# Welcome: What you will build (2 minutes)

In this course you will create a simple roadmap for the next quarter:

- 1 release (e.g., “Q2 2026”)
- 3–5 features in that release
- A features roadmap view you can save and share

## Before you start
You need:
- Access to Aha! Roadmaps
- Contributor permissions in at least one workspace/product (or a demo workspace)

## Suggested time
~55 minutes total (including quiz).
