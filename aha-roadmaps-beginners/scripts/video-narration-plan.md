# Video Narration Plan

## Objective
Create concise narration scripts aligned with each section's learning goal.

## Style guide
- Keep tone clear and instructional
- Prefer short sentences
- Define terms before using them
- Use concrete examples over abstract descriptions

## Per-section script template
For each section, draft:
1. Opening hook (15-30 seconds)
2. Learning objective statement
3. Core walkthrough points
4. Learner action prompt
5. Transition or recap line

## Production checklist
- [ ] Script duration fits target section time
- [ ] UI actions match on-screen steps
- [ ] Terminology is consistent across sections
- [ ] Call-to-action is explicit
- [ ] Final narration reviewed for pacing

## Deliverables
- Narration outline per section
- Final voiceover script per section
- Optional short promo clip script for course landing page
