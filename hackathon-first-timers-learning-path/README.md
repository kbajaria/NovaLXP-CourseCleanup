# Start Here: First-Time Hackathon Pathway

## Purpose
This pathway prepares learners with very low technical confidence to complete the NovaLXP Tech Enablement Track over two days:

1. Agent bootstrapping
2. BigQuery Agent Tools
3. Complaints sub-agent
4. Following Policies with RAG
5. Using MCP with Maps

The path assumes learners may never have coded, used a terminal, worked in a browser-based cloud workstation, or touched Google Cloud before.

## Learning path outcomes
After completing this path, a learner should be able to:

- explain what an AI hackathon is, why organizations run them, and what outcomes teams aim for
- follow step-by-step hackathon instructions without freezing on unfamiliar terms
- use GCP Cloud Workstations and the Google Cloud Console without getting lost
- run basic commands in the browser-based terminal and follow a README or runbook
- understand the basic purpose of an AI agent, a sub-agent, a tool, BigQuery, RAG, and MCP
- complete the two-day track with less facilitator intervention

Each course includes a quiz with an `80%` pass mark. The pathway is complete when all four courses are complete.
