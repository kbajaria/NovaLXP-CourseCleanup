# Testing, debugging, and wrap-up (10 minutes)

## Three simple tests after each major change
1. A normal input
2. A messy or incomplete input
3. An input that should trigger the complaints sub-agent

## Good debugging questions
- Did the app start?
- Did the main agent receive the request?
- Did the sub-agent run?
- Did the output format match what the rest of the app expects?

## Watch for these signs
- no response at all -> app or runtime issue
- generic response -> prompt or routing issue
- wrong data shape -> handoff or parsing issue
