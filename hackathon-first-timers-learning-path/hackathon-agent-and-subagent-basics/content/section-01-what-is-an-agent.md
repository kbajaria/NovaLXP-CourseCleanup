# What an AI agent is and is not (10-12 minutes)

## A useful beginner definition
An AI agent is a piece of software that receives an input, follows instructions, may use tools, and returns an output.

## Simple mental model
- **Input**: the user request
- **Instructions**: the rules and role for the agent
- **Tools**: functions or external services the agent can call
- **Output**: the response or action

## What an agent is not
- not magic
- not always correct
- not the same as a database
- not the same as the application UI

## Why this matters in the hackathon
If you know which part is responsible for what, you can debug more intelligently:
- bad answer -> maybe prompt or tool use
- missing data -> maybe data source or permissions
- broken app -> maybe setup or environment
