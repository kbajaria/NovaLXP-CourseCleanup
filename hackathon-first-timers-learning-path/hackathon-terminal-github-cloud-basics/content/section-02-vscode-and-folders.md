# Cloud Workstations and project folders (12-15 minutes)

## What Cloud Workstations does for you
Cloud Workstations gives you a browser-based development environment inside Google Cloud with:
- a file explorer
- an editor for changing code
- a built-in terminal
- useful visual clues about file names and folder structure

## A good beginner workflow
1. Open the project folder, not just one file.
2. Use the file explorer to confirm the repo structure.
3. Open the terminal inside that same workstation session.
4. Keep the README visible while you work.

## Key concepts
- **Cloud Workstation**: the browser-based coding environment provided in GCP
- **Workspace / project folder**: the folder you opened inside the workstation
- **Explorer**: the left panel showing files
- **Integrated terminal**: the terminal panel inside the workstation editor
- **README**: the file that usually tells you how to set up and run the project

## What to verify before editing
- You are in the correct repo.
- You are editing the intended file.
- You are still inside the correct workstation and project folder.
