# Welcome: Your workbench for the hackathon (3 minutes)

This is **Course 2 of 4** in the **First-Time Hackathon Pathway**.

## Before you start
Complete **Hackathon Pathway 1 of 4: AI Hackathon Orientation** first.

The hackathon instructions may tell you exactly what to type, but you still need a working mental model of:

- what the Google Cloud Console is
- what Cloud Workstations is
- what the browser-based terminal is
- what a repo is inside the workstation
- what a GCP project and environment variable are

This course strips those ideas down to the minimum useful level for a GCP-based hackathon.
