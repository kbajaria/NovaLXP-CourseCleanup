# BigQuery and data retrieval basics (15-18 minutes)

## What BigQuery is
BigQuery is a cloud data warehouse. In simple terms, it is a place to store large structured datasets and query them with SQL.

## Why it shows up in this hackathon
An agent can use tools to retrieve customer or business records from BigQuery instead of inventing answers.

## Beginner mental model
- **Dataset**: a container for tables
- **Table**: rows and columns of data
- **Query**: a request to retrieve or calculate data
- **Result set**: the rows returned by the query

## What beginners usually need to know
- the agent is not the database
- the tool retrieves data from the database
- permissions matter
- if the query returns nothing, that may be a data issue rather than an AI issue
