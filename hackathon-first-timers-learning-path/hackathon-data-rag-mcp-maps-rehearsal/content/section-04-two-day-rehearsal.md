# Two-day rehearsal aligned to the track (15 minutes)

## Day 1 mindset
The first day is about getting the foundation working and learning the project shape.

### Tile 1: Agent bootstrapping
Your goal is to get the starter app running.

### Tile 2: BigQuery Agent Tools
Your goal is to connect the agent to a trustworthy data source and retrieve records.

### Tile 3: Complaints sub-agent
Your goal is to carve out a specialized complaint-handling workflow without breaking the main agent.

## Day 2 mindset
The second day is about grounding and integration.

### Tile 4: Following Policies with RAG
Your goal is to make the agent use policy documents as evidence instead of relying on memory.

### Tile 5: Using MCP with Maps
Your goal is to let the agent call an external map-related service through a structured tool interface.

## Rehearsal checklist
Before the hackathon starts, confirm you can:
- sign into every required tool
- open the repo and README
- open the right GCP project and Cloud Workstations session
- run basic terminal commands in the workstation
- identify where environment values go
- explain BigQuery, RAG, and MCP in one sentence each
- ask for help with the exact step and error
