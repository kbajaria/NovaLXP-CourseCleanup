# Moodle Plugin Settings Template

Set these values in:

`Site administration -> Plugins -> Local plugins -> NovaLXP AI course factory`

## Required

- `Enable front-page course factory` = enabled
- `Lambda function name` = `novalxp-course-factory-<env>`
- `Lambda region` = `eu-west-2`
- `Request timeout` = `120`

## Optional copy

- `Textarea guidance`
- `Submit button text`
