# TalentLMS Migration Course Template

Copy this folder to a new top-level course folder when rebuilding a TalentLMS course for NovaLXP.

Suggested target folder name:
- `<novalxp-shortname>`

After copying:
1. update `course.json`
2. replace placeholder section content in `content/`
3. add or remove sections as needed
4. update `quiz/questionbank.gift`
5. update `quiz/quiz-config.json`
6. add approved media to `media/` if needed

Keep the structure aligned with the existing example course in [aha-roadmaps-beginners/README.md](/C:/Users/kbaja/Projects/novalxp-courses/aha-roadmaps-beginners/README.md).
